# Índice de Registros de Decisiones de Arquitectura (ADR)

Este directorio contiene los "Architecture Decision Records" (ADRs) que documentan las decisiones técnicas más importantes tomadas durante el desarrollo del proyecto Portafolio.

## ¿Qué es un ADR?
Un Registro de Decisión de Arquitectura es un documento corto que captura una decisión arquitectónica importante tomada junto con su contexto y sus consecuencias.
