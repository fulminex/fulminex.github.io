# Arquitectura

## Resumen General

Este proyecto consiste en una aplicación web moderna, estática y renderizada en el cliente (CSR) que funciona como un portafolio de desarrollo. Está diseñada y construida utilizando una arquitectura basada en React que da prioridad a la modularidad, el alto rendimiento y una experiencia de desarrollo (DX) excepcional.

La aplicación utiliza **Vite** como herramienta de empaquetado (build) y servidor de desarrollo, lo que reduce drásticamente los tiempos de inicio y permite una sustitución de módulos en caliente (HMR) altamente eficiente.
